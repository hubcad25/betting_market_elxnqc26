# 🗳️ Pool Élections Québec 2026

Plateforme de prédiction pour les élections québécoises de 2026.

## 🚀 Technologies
- **Frontend:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Backend:** Supabase
- **Icons:** Lucide React

## 🛠️ Installation

```bash
npm install
```

## ⚙️ Configuration
Créez un fichier `.env.local` basé sur `.env.local.example` et remplissez les variables Supabase.

## 🏃 Développement
```bash
npm run dev
```

## 🏗️ Build
```bash
npm run build
```
